# Hash Generator
A simple Python project that generates hashes using different algorithms.

## Features

- Generate MD5 hash
- Generate SHA1 hash
- Generate SHA256 hash

## Technologies

- Python
- hashlib

## Usage

Run:

```bash
python hash_generator.py
```

Then enter any text to generate its hashes.

## Example

Input:

```
Hello
```

Output:

```
MD5
SHA1
SHA256
```
